# Drawescape｜绘梦境 V4

一个把绘画、计算机视觉、透视、建模、色彩和流体变成小游戏的可视化学习平台。

## V4 修复重点

上一版如果 Three.js CDN 没有加载成功，整个 `app.js` 会因为顶层 `import` 失败而停止运行，页面只剩导航，看起来像“按钮不能点 / 后面空白”。V4 已修复：

- `app.js` 不再使用顶层 `import`。
- 默认使用本地可运行的 Canvas 2.5D 引擎。
- Three.js 只在点击“尝试加载 Three.js 实验版”时动态加载；失败也不会影响页面。
- 透视画布的消失点已经可以拖动。
- 画布支持画笔、橡皮、撤销、导入参考图、导出 PNG。
- 建模实验室支持拖动形体、调相机、调 FOV、调光源、显示结构线和投影辅助线。
- 调色画板支持配色生成、明暗/环境/高光通道讲解。
- 风水流体支持风、水、烟、雨四种模拟，拖动金色力场点即可改变方向。

## 本地运行

直接双击 `index.html` 即可运行。  
如果浏览器限制本地文件，也可以在当前文件夹启动一个本地服务器：

```bash
python -m http.server 8000
```

然后打开：

```text
http://localhost:8000
```

## GitHub Pages 部署

1. 新建 GitHub 仓库，例如 `drawescape`。
2. 上传 `index.html`, `styles.css`, `app.js`, `README.md`。
3. 在仓库 Settings → Pages 中选择 `main` 分支根目录。
4. 等待 GitHub Pages 生成网页。

## 目前模块

- 🌌 游戏入口：Drawescape 世界折叠引擎
- 🗺️ 绘境地图：线之海、比例森林、相机塔、深度湖、光之月台、材质星市、风的剧场、流体之谷、折叠云城
- ⭐ 关卡任务：绘画难点拆成小关卡
- 🧊 建模实验室：拖动几何、调相机、调光、看投影
- 📐 透视画布：一点/两点/三点透视，消失点可拖动
- 🎨 调色画板：固有色、亮部、暗部、环境反光、高光
- 💧 风水流体：风、水、烟、雨
