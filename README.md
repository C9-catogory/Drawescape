# Drawescape V8｜Knowledge Blueprint Edition

Drawescape 是一个学习绘画、空间透视、计算机视觉与三维建模知识的可视化探索平台。V8 的重点不是继续堆功能，而是回到知识设计：把绘画难点转译为数学、物理、计算机图形学、计算机视觉和建模中的可解释结构。

## 这版解决什么

之前版本的问题是：首页很酷，但后面的互动、知识和绘画训练没有串起来。V8 改为“知识蓝图优先”：

- 每个模块都按「现实是什么 → 科学如何解释 → 绘画如何压缩 → 如何练习校准」组织。
- 恢复并深化：透视、深度、法线、明暗交界线、材质、褶皱、人脸、自然、风格化。
- 加入 CV / CG / 建模术语星典，把专业词汇转成通俗解释和绘画用途。
- 互动实验使用稳定 Canvas，不依赖外部库，先保证可玩和可理解。

## 模块

1. 绘画信息整合管线
2. Drawescape 世界地图：十个岛屿
3. 跨岛任务：人脸、布料、房间、感受转译
4. 可视化实验室
   - 相机塔：投影几何实验
   - 光之月台：深度 / 法线 / 明暗交界线
   - 褶皱山谷：固定点、张力、重力、材料硬度
   - 人形剧场：头部坐标系与动漫化参数
   - 材质星市：颜色、光照、环境、高光和漫画压缩
   - 风格云城：感受翻译器
5. CV / CG / 建模术语星典
6. 信息整合绘画台

## 运行方式

直接打开 `index.html` 即可运行。也可以用 VS Code Live Server 或部署到 GitHub Pages。

## GitHub Pages 部署

1. 新建仓库，例如 `drawescape`。
2. 上传 `index.html`、`styles.css`、`app.js`、`README.md`。
3. 打开 Settings → Pages。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main / root`。

## 后续建议

V8 是知识设计和稳定交互版。之后不要一次性把所有岛做成复杂物理模拟，而是逐岛深化：

1. 相机塔：真实可拖动消失点、相机、焦距和参考图校准。
2. 光之月台：更细的球、圆柱、人脸局部光影训练。
3. 褶皱山谷：固定点、双点悬挂、肘部压缩、风中飘带。
4. 人形剧场：头骨代理、眼球嵌入、嘴部体块、动漫/Q版参数化。
5. 材质星市：石膏、皮肤、金属、玻璃、布料的分通道可视化。

核心开发规则：任何互动都必须同时回答四个问题：现实中它是什么？计算机如何表示它？绘画中如何压缩它？用户如何练习并校准它？
