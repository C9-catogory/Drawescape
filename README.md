# Drawescape V10｜Navigation Quest Edition

这是 Drawescape 的导航重构版。重点不是增加更多按钮，而是把项目固定成一个可扩展的交互学习游戏空间。

## 核心结构

V10 固定为三层：

1. **世界管线 Pipeline**：现实如何变成画。
2. **知识岛屿 Islands**：十个清晰主问题。
3. **跨岛任务 Quests**：把知识用于真实绘画对象。

## 新增重点

- 游戏 Hub 首页
- 全局导航栏
- 左侧本区总平面 / 关卡树
- 右侧知识检查器
- 快速地图弹窗，按 `M` 可打开
- 十个知识岛屿可独立进入
- 一个完整跨岛任务：画一个盒子房间
- 概念实验室：FOV、深度、明暗、材质、褶皱、风格
- 术语星典：CV / CG / Blender / 绘画术语互译
- 信息整合绘画台
- 我的压缩数据库：保存知识卡、任务卡、实验卡

## 设计原则

每个知识点都必须回答：

- 它属于哪条世界管线？
- 它属于哪个知识岛？
- 它能进入哪个跨岛任务？
- 它有没有互动变量？
- 它有没有反馈标准？
- 它能输出什么卡片？

## 使用方法

直接打开 `index.html` 即可运行。也可以上传到 GitHub Pages。

推荐仓库结构：

```text
index.html
styles.css
app.js
README.md
```
