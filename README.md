# Drawescape 绘梦境 V6｜世界折叠绘画引擎

这是一个面向 ND 群体的可视化绘画学习游戏原型：把绘画拆成坐标、透视、深度、法线、光照、材质、褶皱、自然、风格与情绪表达。

## V6 修复重点

V5 的问题是：首页好看，但后面的知识框架被削弱；人脸、烟、褶皱等过于粗糙；几何体不能删除；材质与颜色/漫反射/高光/粗糙度之间的关系不清楚。

V6 改动：

- 恢复最开始的知识框架：深度图、法线图、透视、渲染通道、信息压缩。
- 3D/2.5D 工作台支持添加、选择、拖动、删除、复制几何体。
- WebGL Three.js 是可选增强；CDN 失败时 Canvas 2.5D 不会崩。
- 材质实验室拆分 Albedo / Diffuse / Specular / Fresnel / AO / Metalness / Roughness。
- 透视学院支持拖动消失点、视平线、绘画辅助线。
- 人脸改为“结构代理模型”，不再假装自动画正确人脸；重点训练中心线、眼线、鼻底、嘴线、转头、动漫化、Q版化。
- 褶皱/自然改为绘画推理器：固定点、张力、材料硬度、风向、流线、烟密度。
- 增加感受翻译器：把情绪翻译成线条、颜色、空间、明暗和构图参数。

## 运行方式

直接双击 `index.html` 可以运行 Canvas 模块。

如果要使用真实 WebGL Three.js，请用 VS Code Live Server 或部署到 GitHub Pages，然后点击“尝试 WebGL / Three.js”。

## 后续方向

- 使用 Three.js TransformControls / OrbitControls，本地化依赖，避免 CDN。
- 加入 GLTF/OBJ 导入与导出。
- 使用更专业的人体/头部代理模型，而不是 2D 简化示意。
- 做真正的布料/头发/水面 shader 或小型物理模拟。
- 将每个模块扩展成关卡：观察 → 扫描 → 推理 → 折叠 → 绘制 → 校准。
- 加入参考文献/课程来源库：透视、图形学、CV、传统绘画、认知科学。

## 项目定位

Drawescape 不是普通绘画教程，而是：

> 一个关于如何看见、推理、折叠现实的绘画探索游戏。

绘画不是复制现实，而是把现实中的结构、光、材质、关系和感受，压缩成一张可以被心理解的平面。
